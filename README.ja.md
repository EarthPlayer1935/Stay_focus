# Stay Focus

ウェブページや画面全体（デスクトップ版）にフォーカス・スポットライトを追加する軽量ツール —— 読んでいる内容に集中するのを助けます。

## 特徴

- 🎯 **スポットライトモード** — 背景を暗くし、カーソル周辺を明るく保ちます
- 🖍️ **ハイライトモード** — 背景は通常通りで、フォーカスエリアにカスタムカラーを表示します
- 📄 **全行モード** — 読書用ルーラーのように、ハイライトが画面幅全体に広がります
- ⬛ 🔲 ⏺ **形状プリセット** — 正方形、角丸矩形、正円をワンクリックで切り替え
- 🔗 **縦横連動 (1:1)** — 幅と高さをロックして、完璧な円や正方形を作成できます
- 🌙 **ナイトモード** — ライト/ダークテーマの切り替え (☀️/🌙)
- ⌨️ **キーボード操作** — ショートカット (`Shift+Alt+方向キー`) による移動やサイズ調整
- 👻 **自動非表示** — マウスがウィンドウを離れたときに自動的に非表示（デスクトップ版は特定プロセスによるフィルタリングに対応）
- ↕️ **調整可能なパラメータ** — サイズ、不透明度、色、角の丸みを自由に調整可能
- 💾 **設定の保存** — セッションを跨いで設定が維持されます
- ⚡ **リアルタイム更新** — ページを更新せずに設定変更が即座に反映されます
- 🖱️ **プレミアム UI** — ホバー効果の改善と全行クリック対応により、より洗練された操作感を提供します

## デスクトップ版 (Windows)

Electron で構築されたデスクトップアプリ版は、ブラウザだけでなくシステム全体で Stay Focus を利用可能にします！

### デスクトップ版の追加機能:
- **グローバルオーバーレイ** — あらゆるアプリ、デスクトップ、ウィンドウ上で動作
- **プロセスバインディング** — 特定のアプリ（プロセスリストで指定）がフォーカスされたときのみ自動で有効化
- **タスクトレイ対応** — トレイアイコンから素早く設定や操作が可能

## 使い方

### ブラウザ拡張機能
1. Chrome に拡張機能をインストールします（`chrome://extensions/` → 展開して読み込む）
2. 拡張機能アイコンをクリックしてコントロールパネルを開きます
3. メインスイッチを切り替えてスポットライトを有効にします

### デスクトップアプリ
1. [リリースページ](https://github.com/EarthPlayer1935/Stay_focus/releases) から最新のインストーラーをダウンロードします
2. `Stay Focus Setup x.x.x.exe` を実行します
3. デスクトップまたはタスクトレイから起動します

## 開発とビルド

このプロジェクトは Node.js を使用してビルドとリリースを自動化しています。

### 拡張機能
1. `npm install`
2. `npm run build` (`release/` ディレクトリに `.zip` が生成されます)

### デスクトップアプリ (Electron)
1. `cd stay_focus_desktop`
2. `npm install`
3. `npm run dist` (`dist/` ディレクトリにインストーラーが生成されます)
