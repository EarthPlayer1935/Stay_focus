# 保持专注 (Stay Focus)

一个轻量级的工具，可以为网页或整个系统桌面（桌面端）添加焦点聚光灯 —— 帮助你更轻松地集中注意力进行阅读。

## 功能特点

- 🎯 **聚光灯模式** — 背景变暗，鼠标周围区域保持明亮
- 🖍️ **高亮模式** — 背景保持正常，焦点区域显示自定义颜色
- 📄 **整行模式** — 高亮区域横跨整个屏幕宽度，像一把阅读尺
- ⬛ 🔲 ⏺ **形状预设** — 一键切换正方形、圆角矩形和圆形
- 🔗 **联动宽高 (1:1)** — 锁定宽度和高度，轻松制作完美的圆形/正方形
- 🌙 **夜间模式** — 支持在明亮和深色 UI 主题间自由切换 (☀️/🌙)
- ⌨️ **快捷键控制** — 使用快捷键快速移动或调整焦点区域 (`Shift+Alt+方向键`)
- 👻 **仅指定程序启用** — 勾选该项即可仅在所选程序上显示聚光灯（其他程序上自动隐藏）
- ↕️ **可调节参数** — 支持调整大小、透明度、颜色和圆角半径
- 💾 **记忆设置** — 自动保存你的偏好设置，下次启动依然有效
- ⚡ **即时生效** — 所有修改实时应用，无需刷新或重新启动
- 🖱️ **精致 UI** — 优化了 Hover 效果和全行点击支持，提供更佳的交互体验

## 桌面端 (Windows)

桌面端应用程序（基于 Electron 构建）将 Stay Focus 的强大功能带到了整个系统，而不仅仅局限于浏览器！

### 桌面端特有功能：
- **全局覆盖** — 可以在任何应用程序、桌面或窗口上方显示
- **进程绑定列表** — 仅在特定的应用程序窗口处于焦点时自动开启或隐藏
- **系统托盘** — 通过托盘图标快速访问设置和控制

## 使用方法

### 浏览器扩展
1. 在 Chrome 中安装扩展程序（`chrome://extensions/` → 加载已解压的扩展程序）
2. 点击扩展图标打开控制面板
3. 切换主开关以启用聚光灯

### 桌面端应用
1. 从 [Releases](https://github.com/EarthPlayer1935/Stay_focus/releases) 下载最新安装程序
2. 运行 `Stay Focus Setup x.x.x.exe`
3. 从桌面图标或系统托盘启动

## 开发与构建

本项目使用 Node.js 来实现自动化构建和发布流程。

### 扩展程序 (Extension)
1. `npm install`
2. `npm run build` (打包后的 `.zip` 位于 `release/` 目录)

### 桌面端 (Electron)
1. `cd stay_focus_desktop`
2. `npm install`
3. `npm run dist` (安装程序位于 `dist/` 目录)

## 常见问题 (FAQ)

### ❓ 为什么在 Windows 任务管理器（或其他高权限窗口）上方聚光灯会卡住？

这是由于 Windows 的 **用户界面权限隔离 (UIPI)** 安全机制导致的。
- **原理**：任务管理器（尤其是开启“总在最前”时）以高完整性级别（管理员权限）运行。为了安全，Windows 不允许普通权限的程序（如 Stay Focus）获取高权限窗口上方的鼠标事件。
- **解决方法**：请尝试**以管理员身份运行 Stay Focus**，这样它将拥有足够的权限在这些窗口上方正常工作。
